# Animal-pro
Aplicativo web diseñado como catálogo comercial para el sector de mascotas.
Pérmite gestionar y visualizar servicios como peluqueria, consultas veterinarias y venta de productos, ofreciendo una experiencia de usuario intuitiva y atractiva.
## Tecnologias
- HTML5
- CSS3
- JavaScript
## Funcionalidades
- Catálogo de servicios para mascotas
- Sección de peluquería
- Información sobre consultas veterinarias
- Visualización de productos disponibles
- Navegación intuitiva para el usuario
